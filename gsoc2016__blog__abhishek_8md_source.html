21 **Project**
22 
23 The project is revolved over integrating object detection and classification module using Convolutional Neural Networks. The following shows the basic components of the work to be completed during the term of Google Summer Of Code, 2016
24 
25  - Implement a way to invoke Caffe open source library from the OpenDetection module with a user-friendly code based way ( this will include a tinge of GUI support for instant access)
26  - Implement open source guidance and codes for state-of-the art object localization problems(hypothesis generation) specifically based on selective-search and convolutional neural network (CNN) approaches.
27  - Adding a ground-truth annotation tool to the module with a graphical-user-interface support.
28  - Implementing short, but effective modules like mixed-pooling, recurrent networks to the Convolutional Neural Networks Training dependent on the invoked caffe library.
29  - Adding context based learning CNNs.
30  - Adding user-interface to train and test CNN based classifiers and object detectors.
31  - Adding documentation for the above
32 
33 All the completed and on-going work will be explained in detail here, as the process moves forward.
