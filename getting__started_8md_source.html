9 ##Install OD {#getting_started1}
10 
11 First install OD as explained in \ref installation_instruction. Basically, after installing dependency you will be downloading and building OpenDetection with:
12 
13 @code{.bash}
14 git clone https://github.com/krips89/opendetection.git
15 
16 cd opendetection
17 mkdir build; cd build
18 cmake ..
19 make -j8
20 @endcode
21 
22 ##Get Data {#getting_started2}
23 
24 After compiling OD, you will need some data to train or some existing pre-trained data to run any of the examples in the `examples` directory. To help get started we have provided our own data in the repo opendetection_data. Download our data from https://github.com/krips89/opendetection_data using
25 
26 @code{.bash}
27 git clone https://github.com/krips89/opendetection_data.git
28 @endcode
29 
30 Make `<path_to_data>/trained_data` as your trained_data location in all the examples, as pre-trained data of some `Detector`s are already provided with proper directory structure.
31 
32 
33 ##Run your first examples {#getting_started3}
34 
35 Go to the build directory of OD (<path_to_source>/build). Run a face detector from your webcam using:
36 
37 @code{.bash}
38 ./examples/objectdetector/od_cascade_cam <path_to_data>/trained_data/
39 @endcode
40 
41 Or get some images containing people (suppose in <path_to_images>/*.JPG) and run the people detector using:
42 @code{.bash}
43 ./examples/objectdetector/od_image_hog_files <path_to_data>/trained_data/ "<path_to_images>/*.JPG"
44 @endcode
45 
46 Depending on the image, you will see something like the following.
