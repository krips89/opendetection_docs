29  /** \brief Detector based on 3D global features like VFH, ESF, CVFH etc.
30  *
31  * This class uses PCL 3d_recognition_framework in the background for the detection. First train your data using ODCADDetectTrainer3DGlobal and use this class for the detection.
32  * This detection will assume the presence of a plane (like a table top) in the pointcloud. It segments the point cloud assuming the presence of a plane and using a simple Euclidian segmentation.
33  * After that it finds the global features of each segmented scene match them with the trained data thereby performing a clssification. Read the documentation of ODCADDetectTrainer3DGlobal to know how
34  * the training data should be arranged and trained to get meaningful detection.
35  *
36  * This class provides a detection for each segmented scenes which matches them the best. So the number of positive detection is same as the number of possible segmented scene in the point cloud.
37  *
38  * \author Kripasindhu Sarkar
39  *
40  */
