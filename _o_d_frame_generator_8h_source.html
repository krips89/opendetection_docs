23  /** \brief The FrameGenerator class for capturing and reading Scenes conveniently.
24  * Templated with two parameters - SceneType identifying a Scene class, and TYPE identifying the type of input. After the instantiation with a correct TYPE, use the function
25  * getNextFrame() to get an instance of next scene of SceneType. getNextFrame() returns valid scenes until all scenes matched are exhausted - the time when 'isValid()' is false.
26  *
27  * \tparam SceneT One of the Scene classes - ODSceneImage or ODScenePointCloud
28  * \tparam TYPE TYPE can be GENERATOR_TYPE_FILE_LIST which means you provide the list of scene files to be returned by the FrameGenerator in the constructor (for eg. \/home/username/pics/*.jpg)
29  * Or it can be GENERATOR_TYPE_DEVICE which picks up the webcam or the kinect based on the SceneType.
30  * \author Kripasindhu Sarkar
31  *
32  */
