Detailed Description

ODHOGMultiscaleDetector: A linear classifier detector for HOG features User can set any linear classifier through setSVMDetector function or one of the default available detectors like OD_DEFAULT_PEOPLE This class will then find HOG features in the scene and use the liniar classification model as an input to classify the scene It covers both simple region detection and omni detection (multiscale mitilocation detection on the entire scene)
