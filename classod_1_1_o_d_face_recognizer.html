Detailed Description

ODFaceRecognizer: global feature based facerecognizer.

Supports EigenFace and FischerFace currently. As an global detector, it does not perform a multiscale omni detection. It just says if a given scene represents a face. To search for a face in an entire scene, you need to use a onnidetector which performs this task on multiscale on each locatoin
